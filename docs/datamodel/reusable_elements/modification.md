# Modification

**Description**: The position where the modification occurs<br/>
**Required**: no <br/>
**Type**: string (fulltext) 

## type 

**Description**: The common name/type of the modification<br/>
**Required**: yes <br/>
**Type**: string (fulltext)

**Required**: no <br/>
**Type**: array of object (string) <br/>
**contains** see [Step](step.md)
