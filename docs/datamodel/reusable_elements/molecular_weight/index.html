Molecular weight

value​

Description: The numerical value of the molecular weight, -1 if unknown
Required: yes
Type: number (float) Minimum: -1

unit​

Description: The unit of the molecular weight
Required: yes
Type: enumerator (string)
Options: ('g/mol', 'Da', 'kDa', 'MDa')
