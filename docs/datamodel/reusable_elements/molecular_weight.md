# Molecular weight

## value 

**Description**: The numerical value of the molecular weight <br/>
**Required**: yes <br/>
**Type**: number (float) 

## unit 

**Description**: The unit of the molecular weight <br/>
**Required**: yes <br/>
**Type**: enumerator (string) 
**Options**: ('g/mol', 'Da', 'kDa', 'MDa')
