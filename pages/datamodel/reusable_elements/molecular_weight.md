# Molecular weight

## value 

**Description**: The numerical value of the molecular weight, -1 if unknown <br/>
**Required**: yes <br/>
**Type**: number (float)
**Minimum**: -1 

## unit 

**Description**: The unit of the molecular weight <br/>
**Required**: yes <br/>
**Type**: enumerator (string) <br/>
**Options**: ('g/mol', 'Da', 'kDa', 'MDa')
